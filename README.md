# Ernährungstracking

Hier werden tägliche Trackingdaten gepflegt, z. B. als `tracking.csv` oder `tracking.xlsx`.

- Zeitformat: ISO (Datum + Uhrzeit), z. B. `2025-07-03 08:15`
- Eine Zeile = eine Mahlzeit oder Snack
- Tagesgrenze: 18:00 Uhr
